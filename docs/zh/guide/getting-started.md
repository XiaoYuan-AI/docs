# 快速开始

欢迎使用小源 AI！本指南将帮助你在喜爱的编程工具中快速上手 Claude 4.8。

## 什么是小源 AI？

小源 AI 通过 OpenAI 兼容的 API 端点提供 Claude 4.8 (Opus) 访问。你可以使用支持 OpenAI API 集成的工具来调用最新 Claude 模型。

## API 端点

```
https://ai.xiaoyuan.co.uk
```

## 支持的工具

- **Claude Code** - Anthropic 官方 CLI 与桌面应用
- **Codex** - AI 编程助手
- **OpenCode** - 开源编程助手
- 任何支持 OpenAI 兼容 API 端点的工具

## 快速开始

1. **获取 API 密钥**，访问 [https://ai.xiaoyuan.co.uk](https://ai.xiaoyuan.co.uk)
2. **选择你的工具**，按照对应的集成指南配置
3. **开始编程**，享受 Claude 4.8！

## 可用模型

配置工具时，可以使用以下模型名称：

- `claude-opus-4-8` - 最新 Claude Opus 模型（推荐）
- `claude-opus-4.8` - 替代格式
- `claude-opus-4-latest` - 始终指向最新版本
